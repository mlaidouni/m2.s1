### Question 2

En mode PWM 8 bits :

- le compteur va de 0 à 255 (TOP = 255)
- La sortie OC0A pase à 0 lorsque TCNT0 atteint OCR0A

Le duty cycle est donc :

$D = \frac{OCR0A}{255}$

| Duty cycle | Calcul     | OCR0A |
| ---------- | ---------- | ----- |
| 0          | 0 x 255    | 0     |
| 25         | 0.25 x 255 | 64    |
| 50         | 0.5 x 255  | 128   |
| 75         | 0.75 x 255 | 191   |
| 100        | 1 x 255    | 255   |
